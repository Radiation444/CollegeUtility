# CollegeUtility

CollegeUtility is a Flutter-based mobile application designed to simplify everyday college activities by providing useful student-focused features in one place.

## Features

### User Profiles
- View and edit profile
- Upload profile photo
- Department and hostel information
- Bio and contact details

### Ride Sharing
- Post rides
- View available rides
- Join rides

### Lost & Found
- Post lost items
- Post found items
- View listings

### Real-Time Messaging
- Real-time chat between users
- Message button on user profiles
- Enable/disable messaging toggle
- Chat UI with message bubbles
- Conversation list

## Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage

## Getting Started

Clone the repository


git clone https://github.com/Radiation444/CollegeUtility.git


Navigate to project


cd CollegeUtility/app


Install dependencies


flutter pub get


Run the app


flutter run
